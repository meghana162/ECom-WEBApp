**AI-Powered E-Commerce Platform**

 1. Overview

AI-Powered E-Commerce Platform is a full-stack web application that combines traditional online shopping functionality with Artificial Intelligence capabilities. The application enables users to browse products, manage shopping carts, place orders, and receive AI-assisted product recommendations and support.

The frontend is developed using React.js, providing a modern, responsive, and interactive user experience. The backend is built with Java and Spring Boot, exposing RESTful APIs for product management, order processing, user interactions, and AI-powered services.

2.  Features

* Product catalog management
* Product search and filtering
* Shopping cart functionality
* Order placement and management
* Product image upload and display
* AI-generated product descriptions
* AI-powered product recommendations
* AI chatbot for customer assistance
* Semantic search using vector embeddings
* Responsive user interface

9. Architecture

The application follows a client-server architecture:

* React frontend communicates with Spring Boot backend through REST APIs.
* Spring Boot handles business logic, AI integration, and database operations.
* PostgreSQL stores application data.
* PGVector stores vector embeddings for semantic search and AI-driven recommendations.
* OpenAI services are integrated through Spring AI for intelligent product descriptions, chatbot responses, and recommendation features.
